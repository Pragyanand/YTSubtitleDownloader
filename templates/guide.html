Why do I need this?

Automating YouTube requires a "driver" that acts as a bridge between this app and your Google Chrome browser. Crucially, the driver version must match your Chrome browser version, or the app will crash immediately.

Step 1: Check your Chrome Version

You need to know exactly which version of Chrome is installed on your computer.

  1. Open Google Chrome.
  2. Click the three dots (menu) in the top-right corner.
  3. Go to Help > About Google Chrome.
  4. Write down the version number (e.g., 121.0.6167.85).

Step 2: Download the Driver

For Chrome 115 and newer

Use the Dashboard

Google now hosts drivers on a dedicated testing dashboard.

Open Dashboard
Look for the "Stable" heading, then download the chromedriver zip for win64 (or win32).
For Chrome 114 and older

Use the Index

Older versions are stored in the legacy index.

Open Legacy Index
Find the folder number that matches your Chrome version exactly.

Step 3: Unzip and Link

Once downloaded:

  • Extract the chromedriver.exe file from the zip.
  • Place it anywhere safe (e.g., directly in this app's folder).
  • In this app, click "Select File" and pick that file.
