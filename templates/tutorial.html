Setup & Tutorial

1. Install Tampermonkey

You need a browser extension to run the automation script.

  • Download Tampermonkey for Chrome

2. Install the UserScript

This script fixes the "multiple download" bug and logs progress to this app.

⬇️ Download Fixed Script (.js)

How to install: Open downloaded file > Copy text > Tampermonkey Dashboard > New Script > Paste > Save.

3. Fetch & Automate

  1. Go to the Home Page.
  2. Enter your API Key and a Channel ID.
  3. Download the Excel file generated.
  4. Go to the Process Page and upload that Excel file.
  5. Select the videos you want to get subtitles for.
  6. Click Start Automation.
